# STM32 G030F6Px
ARM® Cortex®-M0, 32bit, TSOC-20.<br />
Features: 64 MHz MAX, 32K Flash, 8K RAM, `USART`, `SPI`,  `I2C`.<br/>
Board Size: 17.78 mm * 25.4 mm.<br />
Price: about 2,800 KRW, 2.11 USD.
